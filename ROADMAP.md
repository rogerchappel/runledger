# Roadmap

## v0.1

- [x] Record local command runs.
- [x] Append tamper-evident JSONL.
- [x] Summarize and verify ledgers offline.
- [x] Redact obvious secrets by default.
- [x] Ship deterministic fixtures and smoke checks.

## Later

- [ ] Configurable redaction policy from an explicit local file.
- [ ] Optional shell completions.
- [ ] Optional artifact references for files and diffs.
- [ ] Release packaging once distribution policy is approved.

RunLedger should stay small: no hosted dashboard, telemetry, accounts, or default network calls.
